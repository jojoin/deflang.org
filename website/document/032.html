高级类型变量

3 种高级类型：func 函数、proc 处理器、node 节点。所有高级类型同基本类型一样，可作为函数参数、返回值或通过模块导出使用。

node 节点

node 对象是一种比较特殊的对象，通过它可以像处理普通数据一样处理程序代码。用于 block 块结构容器对象下标访问返回值，和调用处理器所传参数。

# node 对象的使用
code : {
　　k
　　print(k)
　　k : k + 1
}
k : 1
eval(code[3])
print(k)　# 2
