关于

Def 是新一代基于虚拟机的开源脚本编程语言，拥有 C 的简捷语法和 Lisp 的强大表现力。

Def 支持多范式、反射、元编程、函数式、垃圾回收等高级特征。通过强大的独一无二的处理器语法，让 Def 成为“可编程的编程语言”，甚至在运行时期改变程序行为。运用其精简直观的语法，可快速构建出模块化、易重构、可读性强的应用程序。

为何使用 Def ?

如果你的报酬是以敲击键盘的次数计算，那么使用 Def 将会让你破产；

如果你想要自由的表达，而不是被语言固有的限制弄得束手束脚焦头烂额；

如果说编程是一次探险，那么你需要一把顺手的瑞士军刀，而不是塞满各种起子和螺丝刀重达30公斤的工具箱；


如何使用 Def ?

不需要安装IDE，不需要下载标准库，不需要环境配置，不需要编译，不需要任何麻烦！

三步即可得到你想要的结果：

　　1. 下载 Def 虚拟机（可执行文件: Windows(64) , Linux(64) , OS X(64)）
　　2. 使用任何文本编辑器编写 Def 代码并保存
　　3. 命令行输入　` def hello.d `　运行代码
