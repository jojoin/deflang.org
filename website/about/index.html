关于 Def

Def 是新一代基于虚拟机的开源脚本编程语言，拥有 C 的简捷语法和 Lisp 的强大表现力。

设计者的理想

Def 是一个注重简捷与自由的语言，它的设计者混合了他喜欢的语言（C、Python 和 Lisp ） 产生了一种灵活独特的具有元编程特性的新语言。在设计 Def 时遵循两点重要的原则：

　　1. 给予使用者最大的自由
　　2. 尽量减少敲击键盘的次数

Def 并不强迫使用者遵循某种“范式”，或者害怕使用者犯错而设下各种禁区，也未对当前的硬件环境做出设计上的妥协。你只需自由的编写，并得到你想要的结果。
