虚拟机下载

我们已经在常用的平台上生成了可用的虚拟机可执行文件，如果你仅仅只是使用 Def 而对其实现不感兴趣，可直接选择对应平台的可执行文件下载：

源码下载

如果没找到你的对应平台的可执行文件，或者像一探 Def 的究竟。你可以直接下载源码。

　　下载源码 zip 包

　　访问 Github 托管页面

Def 的虚拟机采用 C++11 编写，自助编译时需要编译器支持 C++11 标准。
